1：作用
1：生成html代码，input框
2：校验数据

4：有错误提示
2：定义form
from django import forms
class RegForm(forms.Form):
username = forms.CharField(min_length=5,max_length=15,required=True,label=”用户名”,validators=[filter_code],error_messages={
“required”:”不能输入空值”,
“min_length”:”输入长度小于5”,
“max_length”: “输入长度大于15”,
})

使用form

{form_obj.as_p}                    生成所有字段
{form_obj.username}                生成某个字段对应的input标签
{form_obj.username.label}        生成某个字段的标签名
{form_obj.username.id_for_label}生成某个字段的id
{form_obj.username.errors}        生成字段的所有错误信息
{form_obj.username.errors.0}    生成字段的第一个错误信息
{form_obj.errors}                生成表单的错误信息
